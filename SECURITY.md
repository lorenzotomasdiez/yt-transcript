# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability, please **do not open a public issue**.
Instead, email the maintainer privately so the issue can be handled responsibly.

We aim to acknowledge reports within 48 hours and provide a fix or workaround as
soon as possible.
